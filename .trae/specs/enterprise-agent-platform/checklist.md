# 《有机系统》检查清单

> Change-ID: `enterprise-agent-platform`
> 用途：在实现阶段逐项核验，确认需求与架构落地情况。

---

## 平台基座

- [ ] 工程骨架支持 monorepo，本地开发栈（PG/Redis/MinIO/Milvus/Keycloak）可一键启动
- [ ] CI 流水线含 lint/test/build，代码规范生效
- [ ] SSO（OIDC/SAML/LDAP）登录可用，本地账号兜底可用
- [ ] 组织架构同步与角色（employee/skill-author/admin/auditor）正确
- [ ] RBAC + ABAC 权限中间件按部门隔离数据，越权访问被拒绝
- [ ] 模型网关支持多模型路由、限流、配额、成本统计、调用日志
- [ ] 模型健康检查与降级策略生效
- [ ] 对话 API 支持多会话与流式输出（SSE/WebSocket）
- [ ] 短期记忆滑动窗口 + 摘要压缩生效，不超长
- [ ] 对话支持中断、追问、回滚某一步重试
- [ ] Agent 编排引擎支持 Plan→Act→Observe→Reflect 与 Function Calling
- [ ] Agent 执行 trace（输入/工具/思考/输出/耗时/token）可查

## 技能系统与企业内技能市场

- [ ] 技能数据模型完整（元数据/提示词/参数 Schema/工具/工作流/权限范围）
- [ ] 技能 CRUD 与 SemVer 版本管理可用
- [ ] 低代码编辑器支持拖拽工作流 + 提示词编辑 + 在线调试 + 版本回滚
- [ ] 技能运行时沙箱限制文件/网络/内存，恶意代码无法逃逸
- [ ] 技能发布走 admin 审核 + 自动安全扫描（提示词注入/恶意代码）
- [ ] 市场支持分类、关键词+语义搜索、评分、使用量排序
- [ ] 一键安装技能时正确提示依赖 MCP 授权，安装行为入审计
- [ ] 支持紧急下架与已安装版本冻结

## 专家 Agent 市场与预置专家

- [ ] 专家 Agent 由 Persona+技能集+MCP+知识库+记忆策略组成
- [ ] 专家订阅后保留与该员工的独立长期记忆
- [ ] 员工可微调已订阅专家（语气/私有知识）形成个性化变体
- [ ] 市场支持浏览/详情/订阅/评论
- [ ] 运营后台支持上下架与推荐位
- [ ] 预置 6 个专家（投资/法务/HR/会议纪要/文档摘要/数据查询）功能可用

## MCP 集成层与企业系统接入

- [ ] MCP Adapter 框架可暴露 tools/resources/prompts
- [ ] 支持内网纯离线部署，不依赖公网
- [ ] 服务目录展示能力清单、所需权限、数据范围
- [ ] 用户可主动勾选连接、设置授权范围（只读/读写/数据标签）、随时撤销
- [ ] 所有 MCP 调用经网关统一鉴权、脱敏、审计、限流、配额
- [ ] 敏感写操作需二次确认或走审批流
- [ ] 首批 5 类企业系统（OA/CRM/文档/数据中台/邮件日历）Adapter 可用

## 知识库与 RAG

- [ ] 支持多源文档接入（本地/文档系统 MCP/邮件附件/网页白名单）
- [ ] 解析支持 PDF/Word/Excel/PPT/Markdown/图片 OCR
- [ ] 语义分块保留层级与页码
- [ ] 支持多 embedding 模型热切换
- [ ] 元数据标注部门/密级/来源/时间
- [ ] 混合检索（语义+关键词+元数据过滤）返回结果
- [ ] 结果带来源引用（文件名+页码），可跳转原文
- [ ] 未命中时有兜底提示

## 长期记忆与用户画像

- [ ] 事实/偏好/事件三类记忆独立存储，按用户强隔离
- [ ] 记忆写入走"提取-去重-合并"流水线，无冗余
- [ ] 支持时间衰减 + 显式纠正 + 主动遗忘（删单条/清主题）
- [ ] 记忆读写全留痕可审计
- [ ] 偏好学习生效（如连续要求表格输出后默认表格输出）
- [ ] 跨会话记忆召回可自动注入上下文
- [ ] 画像主页可查看/编辑/导出/删除

## 安全合规

- [ ] 传输 TLS1.3、存储 TDE+SSE、向量库加密
- [ ] 企业自管 KMS（支持 HSM）
- [ ] 全链路审计日志覆盖登录/记忆/技能/MCP/Agent/数据访问
- [ ] 审计可按用户/时间/操作检索、导出、告警
- [ ] 脱敏规则可配置（手机号/身份证/金额阈值）
- [ ] 涉密文档访问走审批
- [ ] 数据驻留策略生效（不出指定机房）
- [ ] 提示词注入检测、越权调用检测、输出内容安全过滤可用

## 前端与多端

- [ ] Web 对话工作台支持多会话与侧边栏（记忆/技能/专家/连接）
- [ ] 技能市场与专家市场页面可用
- [ ] 我的管理台（画像/技能/连接/订阅）可用
- [ ] 知识库管理页面可用
- [ ] 管理后台（用户/组织/审计/模型/市场审核）可用
- [ ] Electron 桌面端复用 Web 资源，账号/状态同步
- [ ] Electron 支持全局快捷键、系统托盘、离线缓存
- [ ] 首屏 < 2s；首 token P95 < 1.5s
- [ ] 深色模式、快捷键、中英多语言

## 部署与上线

- [ ] K8s 生产部署高可用（PG/Milvus/MinIO）
- [ ] Prometheus/Grafana/Loki 监控与告警生效
- [ ] 公共技能/专家市场离线镜像可导入
- [ ] 500 并发会话压测通过，首 token P95 < 1.5s
- [ ] 安全渗透测试通过并修复
- [ ] 灰度上线与回滚预案就绪
