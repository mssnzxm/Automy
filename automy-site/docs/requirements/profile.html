    • 用户中心

    用户个人空间,集中管理档案、画像、技能、订阅、MCP 连接、偏好与安全配置。是普通员工日常维护个人信息与掌控自身数据主权的核心入口,也是查看个人记忆画像、管理 API Token 与安全设置的统一界面。

    👤
    模块定位
    本模块定义普通用户视角的全部能力:个人资料(REQ-11.1)、画像主页(REQ-11.2)、我的技能管理(REQ-11.3)、我的订阅管理(REQ-11.4)、我的连接管理(REQ-11.5)、偏好设置(REQ-11.6)。

    模块概述

    Automy 坚持 "用户数据主权归属用户" 原则:用户产生的记忆、偏好、订阅记录均归用户所有,用户可随时查看、编辑、导出、删除。用户中心是这一原则的落地界面,让投资公司员工对自身数据有完全的可见性与可控性。

    用户中心同时承担个性化入口职责:用户在此配置偏好(回复风格、关注赛道、默认模型),这些偏好将影响 Companion 记忆写入、专家订阅个性化、工作台默认参数。所有配置变更实时生效,无需重启服务。

    需求条目

    REQ-11.1 个人资料
    P0

    系统 SHALL 提供个人资料管理界面,展示用户基本信息、联系方式、头像;SHALL 支持头像上传与预设头像选择;SHALL 与 SSO/HR 系统同步权威字段(姓名、工号、部门、职位);SHALL 允许用户编辑非权威字段(联系方式、个性签名)。

    • 基本信息:姓名、工号、部门、职位、邮箱、入职日期(来自 SSO/HR,只读)
    • 联系方式:手机号、企业微信、内部 IM 账号(可编辑)
    • 头像:支持上传(JPG/PNG,≤2MB)或选择预设头像
    • 个性签名:自由文本,≤100 字
    • 资料同步:SSO 登录后自动同步 HR 系统权威字段,字段变更显示"来自 HR"标记
    • 资料可见性:用户可配置哪些字段对同事可见(如手机号仅本部门可见)

    场景

    WHEN 投研部研究员张三入职后首次登录 Automy
    THEN 系统通过 SSO 同步 HR 信息,张三的姓名、工号 EMP-2026-0123、部门"投研部·新能源组"、职位"高级研究员"、邮箱 zhangsan@corp.com 已自动填充且只读;张三上传个人头像,补充企业微信 zhangsan_wx,设置个性签名"专注新能源赛道研究",保存后资料卡片在同事查看时正确展示。

    REQ-11.2 画像主页
    P0

    系统 SHALL 提供画像主页,可视化展示用户的长期记忆画像,按事实、偏好、事件三类分组展示;SHALL 支持单条记忆的查看、编辑、删除;SHALL 支持导出画像为 JSON / Markdown;SHALL 支持按类型、时间、来源筛选记忆条目。

    记忆类型展示形式典型示例
    事实卡片列表(属性 + 值)职位:高级研究员 / 团队:新能源组
    偏好标签云 + 详情关注赛道:新能源、半导体 / 回复风格:详细
    事件时间线2026-07-10 完成某新能源公司财报分析
    • 三类可视化:事实(卡片网格)、偏好(标签云)、事件(时间线),直观展示用户画像
    • 单条编辑:点击任意记忆条目,可编辑内容、修改类型、添加备注
    • 批量删除:勾选多条记忆,一键删除(二次确认)
    • 导出画像:导出为 JSON(完整结构)或 Markdown(可读报告),含时间戳与签名
    • 来源追溯:每条记忆展示来源(对话/手动/技能写入)与写入时间
    • 记忆统计:展示记忆总数、各类占比、近 30 天新增趋势
    • 清除范围:支持按时间范围、类型、来源批量清除记忆

    场景

    WHEN 张三想查看自己的完整画像并导出备份
    THEN 张三进入画像主页,看到三类记忆:事实 18 条(职位、团队、工号等)、偏好 12 条(关注赛道:新能源、半导体、储能;回复风格:详细;默认报告格式:PDF)、事件 45 条(近期研究报告、会议纪要处理);点击"导出 Markdown",系统生成包含时间戳与用户签名的画像报告下载到本地。
    WHEN 张三发现画像中有一条错误偏好"关注赛道:房地产"(系误判写入)
    THEN 张三在该条记忆上点击"编辑",将类型改为"已忽略"或直接点击"删除",二次确认后该条记忆从画像中移除,后续 Companion 不再基于该偏好生成回复;操作记录到记忆审计日志。

    REQ-11.3 我的技能管理
    P0

    系统 SHALL 提供"我的技能"管理界面,展示用户已安装的技能列表;SHALL 支持技能更新、卸载、配置;SHALL 展示技能使用统计与版本信息;SHALL 对卸载操作进行依赖检查与二次确认。

    • 已安装列表:展示技能名称、版本、来源(市场/团队/自创)、安装时间、最近使用时间
    • 技能更新:检测新版本,一键更新;展示版本变更说明
    • 技能卸载:卸载前检查依赖(是否有专家订阅依赖该技能),有依赖时提示用户
    • 技能配置:为技能配置参数(如"财务报表解析"的默认会计准则、报告模板路径)
    • 使用统计:展示每个技能的调用次数、成功率、平均耗时、最近 30 天趋势
    • 技能启停:临时禁用某个技能(不卸载,暂停使用)
    • 权限审查:展示技能声明的权限点(文件访问、网络调用、MCP 工具调用)

    场景

    WHEN 张三不再使用"会议纪要模板"技能,希望卸载
    THEN 张三在"我的技能"找到该技能,点击"卸载";系统检查依赖,发现无专家订阅依赖该技能,弹出二次确认"卸载后将无法在会话中调用,确定卸载?";张三确认后技能卸载,操作记录到审计日志;若该技能被某专家订阅依赖,系统提示"该技能被已订阅专家'行政助理'依赖,卸载前请先取消相关订阅"。

    REQ-11.4 我的订阅管理
    P0

    系统 SHALL 提供"我的订阅"管理界面,展示用户已订阅的专家 Agent 列表;SHALL 支持取消订阅、查看订阅详情;SHALL 展示订阅使用统计(调用次数、Token 消耗、成本);SHALL 提供账单明细与历史账单查询。

    • 已订阅专家列表:展示专家名称、Persona、订阅时间、个性化状态
    • 订阅详情:查看专家绑定的技能集、MCP 服务、知识库、记忆策略
    • 取消订阅:取消订阅后专家不再出现在工作台快捷入口,但历史会话保留
    • 个性化配置:重新配置订阅时的个性化参数(如关注赛道、回复风格)
    • 使用统计:按月展示每个订阅专家的调用次数、Token 消耗、平均响应时长
    • 成本明细:展示本月各订阅产生的成本,按部门配额分摊展示
    • 历史账单:查询历史月份的账单明细,支持导出 Excel

    场景

    WHEN 张三想查看本月专家订阅的使用情况与成本
    THEN 张三进入"我的订阅",看到已订阅 3 个专家:"投研助理"(调用 45 次,¥128)、"法务顾问"(调用 12 次,¥45)、"合规审查员"(调用 8 次,¥32);总成本 ¥205,在部门配额 ¥500 内;张三发现"法务顾问"使用较少,考虑取消订阅,点击"取消订阅",二次确认后取消,下月不再计费。

    REQ-11.5 我的连接管理
    P0

    系统 SHALL 提供"我的连接"管理界面,展示用户已授权连接的 MCP 服务列表;SHALL 支持查看授权范围、撤销授权;SHALL 展示连接使用统计;SHALL 在撤销授权前进行依赖检查(是否有专家/技能依赖该连接)。

    • 已连接 MCP 列表:展示服务名称、连接时间、授权范围(工具列表、数据范围)
    • 授权详情:查看该连接授予的工具调用权限、数据访问范围、写入权限
    • 撤销授权:撤销前检查依赖,若有专家/技能依赖该连接,提示用户先处理依赖
    • 使用统计:展示每个连接的调用次数、调用工具、Token 消耗、最近调用时间
    • 权限审查:定期提示用户审查已授权连接,确认是否仍需要(默认每 90 天提示)
    • 连接日志:查看该连接的调用历史(只展示元数据,不展示具体数据内容)
    • 风险提示:对高权限连接(如写操作、敏感数据访问)标记风险等级

    场景

    WHEN 张三不再需要访问 Wind 数据 MCP 服务,希望撤销授权
    THEN 张三在"我的连接"找到"Wind 数据"连接,查看授权范围(行情查询、财务数据查询),点击"撤销授权";系统检查依赖,发现"投研助理"专家依赖该连接,提示"撤销后'投研助理'将无法调用 Wind 数据,是否继续?";张三确认后授权撤销,该连接的所有 Token 失效,操作记录到 MCP 审计日志。
    WHEN 系统检测到"客户 CRM"连接已 120 天未调用
    THEN 系统在用户中心首页与邮件中提示张三"连接'客户 CRM'已 120 天未使用,建议审查并撤销授权以降低风险";张三点击提示进入连接管理,确认后撤销该连接。

    REQ-11.6 偏好设置
    P1

    系统 SHALL 提供偏好设置界面,允许用户配置主题、语言、默认模型、回复风格、关注领域、通知设置;SHALL 实时生效,无需重新登录;SHALL 将偏好同步到 Companion 长期记忆与专家订阅个性化配置。

    偏好类别可选项影响范围
    主题浅色 / 深色 / 跟随系统全站界面主题
    语言简体中文 / English界面语言
    默认模型从可用模型列表选择新建会话默认模型
    回复风格简洁 / 详细 / 正式 / 轻松Companion 与专家回复风格
    关注领域赛道(新能源/半导体/医药...)+ 业务方向Companion 偏好记忆、专家个性化
    关注条款法务相关重点关注条款法务顾问专家个性化
    默认参数估值模型、报告格式、会计准则技能默认参数
    通知设置邮件 / 企业微信,频率(实时/日报/周报)系统通知渠道
    • 实时生效:偏好保存后立即生效,无需重新登录或刷新页面
    • 偏好同步:回复风格、关注领域同步写入 Companion 长期记忆的偏好类
    • 专家个性化:关注领域、关注条款同步到已订阅专家的个性化配置
    • 偏好继承:部门级默认偏好(如投研部默认关注新能源),用户可覆盖
    • 偏好导入:支持从同事导出偏好配置(JSON)快速导入
    • 偏好重置:一键重置为系统默认偏好

    场景

    WHEN 张三希望切换深色主题并将默认模型改为成本更低的内部模型
    THEN 张三在偏好设置中选择主题"深色",界面立即切换为深色;在"默认模型"下拉中选择"内部模型-研究型"(自部署,成本为商用模型的 1/10);保存后新建会话默认使用该内部模型,预计本月个人成本从 ¥128 降至 ¥18;偏好变更同步到 Companion 记忆。
    WHEN 张三配置关注领域为"新能源、储能、半导体"
    THEN 系统将该偏好写入 Companion 长期记忆偏好类;后续张三与"投研助理"专家对话时,专家主动推荐这三个赛道的最新研报;偏好也同步到画像主页的偏好标签云。

    安全设置(附加)

    用户中心还提供以下安全配置入口,与 安全合规模块 联动:

    • 修改密码:定期修改密码(同步 SSO 密码策略)
    • 多因素认证:开启 TOTP 二次验证(支持企业微信扫码)
    • 登录设备:查看与管理登录设备,支持远程下线
    • 登录历史:查看最近 30 天登录记录(IP、设备、时间)
    • API Token:创建、权限范围、有效期、使用记录、撤销(详见 REQ-8.3 第三方工具扩展)

    • ✅ 我的技能支持更新、卸载(含依赖检查)、配置、使用统计
    • ✅ 我的订阅支持取消订阅、使用统计、成本明细、历史账单
    • ✅ 我的连接支持授权范围查看、撤销(含依赖检查)、定期审查提示
    • ✅ 偏好设置实时生效,同步 Companion 记忆与专家个性化
