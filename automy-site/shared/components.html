    什么是 Automy?
    Automy 奥托迈 是一个企业级自主智能体平台,面向投资公司内部使用, 支持长期记忆、技能市场、专家 Agent、MCP 集成等核心能力。
    支持哪些部署方式?
    支持完整私有化部署到企业内网,模型可对接本地推理,数据全链路不出企业边界。
    如何接入企业系统?
    通过 MCP(Model Context Protocol)标准化协议接入企业 CRM、ERP、文档库等系统, 用户主动连接,可控可审计。

    代码块 Code Block

    带语法高亮占位与复制按钮

    JavaScript 
    // Automy 智能体调用示例
import { Automy } from '@automy/sdk';

const agent = new Automy({
  endpoint: 'https://automy.internal.corp',
  token: process.env.AUTOMY_TOKEN,
  memory: { scope: 'user' }
});

const result = await agent.run({
  expert: 'investment-analyst',
  input: '分析本季度 IPO 项目风险点',
  tools: ['crm.query', 'knowledge.search']
});
